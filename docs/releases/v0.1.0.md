# GitHub Release Draft: v0.1.0

このファイルは、GitHub Releaseへ転記するための下書きです。0.1.0はChrome Web Storeで一般公開済みです。

## タイトル

v0.1.0 - Chrome拡張としての初期公開版

## 本文

AIまえチェック v0.1.0 は、ChatGPT / Claude / Geminiへ文章を貼り付ける前・送信する前に、個人情報・秘密情報・APIキーなどの消し忘れに気づくためのChrome拡張初期公開版です。

Chrome Web Store:
<https://chrome.google.com/webstore/detail/idedmkfplfieijdcflcogkngplhkkecc>

### 主な機能

- ChatGPT / Claude / Geminiでの貼り付け前チェック
- 送信ボタン・Enter送信前の確認
- メールアドレス、電話番号、JWT、AWS Access Key風文字列、GitHub token風文字列、秘密鍵、.env形式の秘密情報、Basic認証URL、クレジットカード風番号などのルールベース検出
- 高リスクまたは秘密情報保護対象の安全化なし送信抑止
- 日本語ラベルによる安全化
- WebLLMによるブラウザ内AI文脈チェック
- Options Pageでの対象サイト・検出ルール・WebLLM設定
- Cloudflare Pages上の紹介LP、プライバシーポリシー、サポートページ

### プライバシー

- 貼り付け本文や送信本文は永続保存しません
- 設定のみ `chrome.storage.local` に保存します
- 外部LLM APIや開発者のサーバーへ本文を送信しません
- WebLLMの初回利用時には、ローカル推論用のモデルファイルを取得する場合があります

### 既知の制限

- 情報漏洩を完全に防ぐものではありません
- 検出漏れや誤検出が発生する可能性があります
- WebGPU非対応環境ではAI文脈チェックを利用できない場合があります
- 0.1.0では署名付きルール配信の本番有効化は見送り、同梱ルールを主に使います

### 次の予定

- 0.1.1でルール配信署名鍵を再発行し、Cloudflare本番Secretへ反映
- Perplexity adapterの追加検証
- 拡張モーダルのプロダクトUI再設計
